## 1. 条件筛选

使用 `loc` 进行条件筛选。

##### Python3

```python
import pandas as pd

def find_products(products: pd.DataFrame) -> pd.DataFrame:
    return products.loc[(products['low_fats'] == 'Y') & (products['recyclable'] == 'Y'), ['product_id']]
```
