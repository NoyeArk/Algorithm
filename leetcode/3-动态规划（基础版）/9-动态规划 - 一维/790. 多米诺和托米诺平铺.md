> **作者:** 弘树

## 解题思路
### 1.动态规划

状态表示：$f[i]$表示可以平铺$2 \times n$的面板的方法的数量

状态计算：怎么消除中间一些重复的状态

这个规律是怎么找到的？

- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$

```C++
class Solution {
public:
    long long f[1010], mod = 1e9 + 7;

    int numTilings(int n) {
        f[1] = 1, f[2] = 2, f[3] = 5;
        for (int i = 4; i <= n; i ++)
            f[i] = 2 * f[i - 1] + f[i - 3], f[i] %= mod;
        return f[n];
    }
};
```
