## 1. 动态规划

### 状态表示

$f[i][j]$ 表示到达第 $i$ 行第 $j$ 列的走法有多少种

### 状态计算

每个状态都可以从上方或者左边移动过来，所以状态转移如下：

$$
	f[i][j] = f[i - 1][j] + f[i][j - 1]
$$

- 时间复杂度: $O(mn)$
- 空间复杂度: $O(mn)$

class Solution {
public:
    int f[105][105];
    int uniquePaths(int m, int n) {
        f[0][1] = 1;
        for (int i = 1; i <= m; i ++)
            for (int j = 1; j <= n; j ++)
                f[i][j] = f[i - 1][j] + f[i][j - 1];
        return f[m][n];
    }
};
